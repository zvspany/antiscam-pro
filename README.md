# AntiScam Pro

Aplikacja webowa do sprawdzania podejrzanych wiadomości, numerów telefonów i linków.

## Uruchomienie

1. Stwórz wirtualne środowisko:
   python3 -m venv venv
   source venv/bin/activate

2. Zainstaluj zależności:
   pip install -r requirements.txt

3. Uruchom aplikację:
   python app.py
